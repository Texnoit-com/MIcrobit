Фоторезистор
------------

:download:`Скачать файл с классов <Photocell.py>`

**Фоторезистор** — компонент, меняющий сопротивление в зависимости от количества света падающего на него.
В полной темноте он имеет максимальное сопротивление, а по мере роста освещённости сопротивление 
уменьшается.

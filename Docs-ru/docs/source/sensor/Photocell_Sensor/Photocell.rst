Фоторезистор
------------

:download:`Скачать файл с классов <Photocell.py>`

**Фоторезистор** — компонент, меняющий сопротивление в зависимости от количества света падающего на него.
В полной темноте он имеет максимальное сопротивление, а по мере роста освещённости сопротивление 
уменьшается.

.. image:: Photocell_Sensor.png
    :width: 100px
    :align: center

Класс
*****

.. py:class::
    Photocell

Класс используется для определения объектов, имеющих поведение датчика фоторезистор.
    
Пример объявления объекта::

    photo= Photocell(pin0)

.. py:function:: photo.get_signal()

Команда позволяет получить аналоговый сигнал с датчика

Датчик окружающего света
------------------------

:download:`Скачать файл с классов <TEMT.py>`

Датчик чувствителен ко всему световому диапазону видимого света. Если уровень света очень
низок, то уровень напряжения на выходе SIG будет также очень мал. При увеличении
освещенности на выходе также будет увеличиваться напряжение.

.. image:: TEMT6000AmbientLight.png
    :width: 100px
    :align: center

Класс
*****

.. py:class::
    ATEMT6000_AmbientLight

Класс используется для определения объектов, имеющих поведение светочуствительного датчика.
    
Пример объявления объекта::

    temp=TEMT(pin0)

.. py:function:: photo.get_signal()

Команда позволяет получить аналоговый сигнал с датчика

Датчик давления
---------------

:download:`Скачать файл с классов <Thin-film_Pressure.py>`

Датчик использует гибкий чувствительный к давлению материал с ультратонкой пленкой. Он водонепроницаем 
и чувствителен к давлению.

Когда датчик определяет внешнее давление, сопротивление датчика изменится. Затем через схему можно преобразовать изменения сопротивления в изменения напряжения, вывести его на конце сигнала.
Таким образом, мы можем получить условия изменения давления, обнаружив изменения сигнала.

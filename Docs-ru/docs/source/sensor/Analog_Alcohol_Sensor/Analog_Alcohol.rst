Класс
*****

.. py:class::
    Analog_Alcohol

Класс используется для определения объектов, имеющих поведение датчика испарения спиртов
    
Пример объявления объекта::

    alcohol=Analog_Alcohol(pin0)

.. py:function:: alcohol.calibrate()

Команда позволяет провести каллибровку датчика. Прогревает датчик в течении 20 секунд.
Далее Вы вращаете подстроечный резистор, пока не погаснет красный светодиод.
Пробуете поднести датчик к испарению спирта - диод должен загореться. 
Убираете датчик от испарения - диод гаснет.
Подтверждаете калибровку, удерживая **кнопку "А"**. Выводится показание датчика.

.. py:function:: alcohol.get_signal()

Команда позволяет получить аналоговый сигнал с датчика

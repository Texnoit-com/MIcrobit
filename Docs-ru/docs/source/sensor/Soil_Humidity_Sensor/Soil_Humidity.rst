Датчик влажности
----------------

:download:`Скачать файл с классов <Soil_Humidity.py>`

Принцип работы датчика - если в почве не хватает воды, аналоговое значение, выводимое датчиком, уменьшится, 
в противном случае оно увеличится.

.. image:: Soil_Humidity_Sensor.png
    :width: 100px
    :align: center

Класс
*****

.. py:class::
    Soil_Humidity

Класс используется для определения объектов, имеющих поведение датчика влажности почвы.
    
Пример объявления объекта::

    soil= Soil_Humidity(pin0)

.. py:function:: soil.get_signal()

Команда позволяет получить аналоговый сигнал с датчика

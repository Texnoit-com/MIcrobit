Класс
*****

.. py:class::
    Driver_Motor

Класс используется для определения объектов, имеющих поведение **трехколесного робота** с пассивным колесом. 
Класс следует применять при подключении моторов к драйверу двигателей. Драйвер должен иметь 6 управляющих пинов. 
4 пина указывают направления и 2 пина указывают мощность мотора.
    
Пример объявления объекта::

    l_motor=Motor(pin13,pin14,pin0)
    r_motor=Motor(pin15,pin16,pin1)

    car = Driver_Motor(l_motor,r_motor)

Вы объявляете объекты левого и правого мотора, которые подключены к драйверу. Передаете их параметрами
в объект класса **Driver_Motor**.

.. py:function:: car.forward(speed:int)

    Команда указывает роботу ехать вперед. **speed** - задает скорость вращения.

.. py:function:: car.backward(speed:int)

    Команда указывает роботу ехать назад. **speed** - задает скорость вращения

.. py:function:: car.stop()

    Команда останавливает робота.

.. py:function:: car.left(speed:int)

    Команда указывает роботу повернуть налево. **Левое колесо-отключено**.

.. py:function:: car.right(speed:int)

    Команда указывает роботу повернуть направо. **правое колесо-отключено**.

Танковый разворот
*****************

.. py:function:: car.left_tank(speed:int)

    Команда указывает роботу повернуть налево. 

.. py:function:: car.right_tank(speed:int)

    Команда указывает роботу повернуть направо.

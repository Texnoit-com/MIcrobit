BBC micro:bit MicroPython - на русском
======================================

Эта документация включает уроки для учителей и документацию по работе с основными датчиками, с которыми вы можете столкнуться при обучении детей.
Надеемся, Вам понравится работать с BBC micro:bit с использованием MicroPython и нашим вспомогательным материалом.

Сервопривод
-----------

:download:`Скачать файл с классов <Micro_Servo.py>`

.. image:: Micro_Servo.png
    :width: 200px
    :align: center

Для работы сервопривода мы должны послать ему импульсный сигнал частотой 50 Гц (Гц). 
Это стандарт почти для всех серводвигателей постоянного тока. Импульсный сигнал выглядит так:

.. image:: servo1.png
    :width: 200px
    :align: center

50 Гц означает, что один импульс происходит 50 раз в секунду. Если разбить его по-другому, 
один импульс отправляется каждые 1/50 секунды , что равно 20 миллисекундам (сокращенно мс).

.. image:: servo2.png
    :width: 200px
    :align: center

Класс
*****

.. py:class::
    Micro_Servo

Класс используется для определения объектов, имеющих поведение серво-моторов.
    
Пример объявления объекта::

    servo=Micro_Servo(pin0)

.. py:function:: servo.angle(arg:int)

    Команда перемещает сервомотор в указанный угол.
